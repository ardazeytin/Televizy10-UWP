# Televizy10-UWP
Windows 10 Universal Windows Platform application for watching TV. Working on Windows 10 desktops, tablets and mobile.

## Built With
* [Windows 10 SDK] 
* [Visual Studio 2015]
* [Microsoft Emulator for Windows 10 Mobile]
